# 论文辅助阅读 Agent P0 规格（MVP）

## 目标

在单篇论文阅读场景下，提升高频问答效率，减少重复输入上下文。

## 快捷动作

- `3分钟速读`：输出研究问题、方法、核心贡献。
- `章节摘要`：按论文结构输出每章要点。
- `术语解释`：解释关键术语并给出语境化说明。
- `关键结论提取`：提取可直接记笔记的结论要点。

## 输入上下文

- `paperId` / `paperTitle`：当前选中文献。
- `currentPage`：用户正在阅读页码。
- `selectedText`：用户圈定的段落文本。
- `quickAction`：当前触发的快捷动作 ID。

## 响应模板

所有回答默认遵循以下结构：

1. `【要点速览】`：3-5 条结论。
2. `【正文解读】`：围绕用户问题展开解释。
3. `【阅读建议】`：1-3 条下一步动作。

## 索引状态可见性

每篇文献在文献库列表显示索引状态：

- `待索引`
- `索引中`
- `可问答`
- `索引失败`
